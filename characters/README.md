# Characters 1206 v2

Чистая структура персонажей, как в Академии.

## Правило структуры

- `main.yaml` — паспорт, отображение имени, внешность, короткие ссылки.
- `character.yaml` — характер, голос, привычки, энергия, текущие цели, правила сцены.
- `past.yaml` — прошлое/hidden/источники знаний, только если прошлое реально важно для сцен.

## Запрет

Не создавать:
- `characters/main/*.md`
- `knowledge/characters/*.md`
- отдельные canon-файлы под способности персонажей
