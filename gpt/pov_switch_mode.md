# POV Switch Mode — 1206 v2

Этот файл описывает только технический режим смены POV.

Режим включается **только** если игрок явно написал в последнем ходе:

- `пов: <имя>`
- `POV: <name>`

## Общие правила

- POV по умолчанию — Акира.
- `пов: Акира` ничего не меняет и считается обычным ходом.
- Для 1206 разрешён не-Akira POV только для целей теста/сцены: `Райден`, `Рэй`, `Ирэй`, `Хару`.
- Если указан неизвестный POV, сцена не должна выдумывать нового POV-персонажа.

## Как читать ввод

- Текст **вне скобок** — точная речь POV-персонажа.
- Текст **в скобках** — действие / намерение / телесная реакция POV-персонажа.
- Нижние блоки `Что ... могла бы сказать` и `Мысли ...` относятся к текущему POV-персонажу, а не к Акире.

## Ограничения знания

- Смена POV не даёт Акире знания автоматически.
- Акира узнаёт что-либо только если она реально увидела, услышала, была ей это сказано, или это сохранено в knowledge/state.
- То же правило действует и в обратную сторону: POV-персонаж знает только то, чему у него есть источник.

## Шапка сцены

Если активен не-Akira POV, после обычной шапки сцены добавить строку:

```text
🎥 POV: {visible_name} · фокус сцены не Акира
```

## Формат сцены

- Сохраняются все обычные правила 1206: граница действия игрока, незакрытый крючок, NPC boundary, календарные ограничения, нижний блок, состояние и отношения.
- В `Отношениях` и `Состоянии` допустимо показывать POV-персонажа как текущий фокус, но не ломать систему total scores.
- Если POV-персонаж напрямую обращается к Акире, сцена всё равно не должна решать за Акиру её ответ/игнор/уход.
